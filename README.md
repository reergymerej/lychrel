# lychrel

## isPalindrome(val)
* @param {String/Number} `val`
* @return {Boolean}

```js
import {isPalindrome} from 'lychrel';

isPalindrome(123321);  // true
isPalindrome('otto');  // false
```


## reverseAndAdd(val)
* @param {Number} `val`
* @return {Number}

```js
import {reverseAndAdd} from 'lychrel';

reverseAndAdd(14);  // 55
reverseAndAdd(123);  // 444
```
